Sensor Data Management From Hardware To Cloud.

The goal of the project is to understand how data can flow between a number of different devices

Using the accelerometer on the F4 Discovery microcontroller we read a few seconds of data. This data is then sent over to the Nucleo board. Once it is received it is transfered to a mobile device using Bluetooth Low Energy (BLE). This device then communicates with Amazon Web Service (AWS) and stores the data into an S3 bucket. A lambda function is then used to convert the acceleration data into the pitch and roll of the F4 Discovery board.
