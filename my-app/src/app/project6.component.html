Mandelbrot Set Renderer

The MandelBrot Set represents all the points on the complex plane that remain bounded when plugged into the following recurrence relation:

Zn+1 = Zn2 + c

Where:

Z0 = c = x + iy

For efficiency purposes we select a threshold (usually 2) and we assume that if this threshold is exceeded, then the sequence for that specific number diverges to infinity. As a result we color this point based on the number of iterations it took to go over the limit. If after a certain number of iterations, the threshold is not reached, we color the point in black.

The renderer allows the user change the maximum number of iterations, the threshold, the light coefficient and the color scheme. The user can also zoom in and out.
